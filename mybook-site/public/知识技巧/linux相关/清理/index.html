    清理

    • 清理 APT 安装缓存
    • 删除无用依赖包
    • 清理 snap 缓存（旧版本 .snap 文件）
    • 清除旧系统日志
    • 删除老的 log 文件备份
    • 清理/tmp 下的临时文件

    清理 APT 安装缓存 #

    sudo apt clean

    删除无用依赖包 #

    sudo apt autoremove –purge

    清理 snap 缓存（旧版本 .snap 文件） #

    sudo rm -rf /var/lib/snapd/cache

    清除旧系统日志 #

    sudo journalctl –vacuum-size=100M

    删除老的 log 文件备份 #

    sudo rm -rf /var/log/.gz /var/log/.[0-9]

    清理/tmp 下的临时文件 #

    sudo rm -rf /tmp/*
